# Miss Universe Philippines 2026 Fan Ballot

A React + Vite + Tailwind CSS website for MUPH 2026 fan crown picks.

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Deploy to GitHub Pages

1. Change `YOUR_USERNAME` in `package.json`.
2. If your repo name is not `muph-2026`, update `base` in `vite.config.js`.
3. Run:

```bash
npm run build
npm run deploy
```
